# Student Management System

This is a simple Student Management System implemented in Java, designed to manage student records with basic operations such as adding, updating, displaying, and deleting student information.

## Overview

The project consists of several Java files:

- `Student.java`: Defines the `Student` class with attributes such as ID, name, and age, along with getters and setters to store and retrieve student details.

- `StudentUtility.java`: Provides utility methods to interact with student records. It allows users to input student details, update existing records, display all student records, and delete specific student records.

- `StudentTest.java`: Contains the main method for running the system. It provides a simple user interface with options to add a student, update a student, display all students, delete a student, and exit the system.

Additionally, the project includes some other files such as `.classpath`, `.project`, and `.settings`, which are configuration files for an IDE like Eclipse.

## Usage

To run the Student Management System, follow these steps:

1. Compile the Java files:
javac src/StudentManagement/*.java


2. Run the application:
